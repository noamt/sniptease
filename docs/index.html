Three steps. No setup.

SnipTease lives in your menu bar. It's always there when you need it.

1

Hit the hotkey

Press ⌃⇧S from any app to open the capture overlay.

2

Pick a preset

Tab through LinkedIn, X, Instagram, OG — or click one in the menu bar dropdown.

3

Drag & capture

Select a region. The framed screenshot lands on your Desktop and clipboard, ready to paste.

What you get

Native, tiny, no Electron. Pure Swift and SwiftUI using ScreenCaptureKit.

⌨

Global hotkey

⌃⇧S from any app. No need to switch windows or open a menu first.

▣

Safe-zone guides

Optional margin overlay shows each platform's content-safe area so nothing important gets clipped.

🖼

Retina-correct export

Captured at your display's native resolution, then Lanczos-downscaled to the exact export size the platform wants.

🤖

MCP server for agents

Run with --mcp and your AI agent can take framed screenshots using natural language descriptions.

For agents, too

SnipTease doubles as a Model Context Protocol server. Your agent describes what to capture in plain English — SnipTease asks a vision model for the bounding box, frames the result, and saves it.

"Capture the authentication section of the README for a LinkedIn post."

// Add to your agent's MCP config (Claude Desktop, Cursor, etc.) { "mcpServers": { "sniptease": { "command": "/Applications/SnipTease.app/Contents/MacOS/SnipTease", "args": ["--mcp"], "env": { "GEMINI_API_KEY": "your-key-here" } } } }
list_presets

Returns the full preset catalog with aspect ratios and export sizes.

agent_capture

Takes a screenshot, finds the described region via VLM, frames it, saves PNG + copies to clipboard.

Build from source

macOS 14+ Xcode 15.4+
git clone https://github.com/noamt/sniptease.git cd sniptease open SnipTease.xcodeproj # Press Cmd+R, or from the command line: xcodebuild -project SnipTease.xcodeproj \ -scheme SnipTease -configuration Release \ CODE_SIGN_IDENTITY="" CODE_SIGNING_REQUIRED=NO \ CODE_SIGNING_ALLOWED=NO build

Building from source produces an unsigned binary. On first launch, right-click and choose Open, or run xattr -dr com.apple.quarantine SnipTease.app.
